   Demo of using Java Spring Dynamic Jasper to create an excel with a big number of records

   Requirements:
   
   Java JDK, maven
   
   from maven execute clean package or clean compile or clean install so maven can take it's dependencies
   and create the target folder

   Run XlsxReportTest.java form the junit runner.
   
   Wait for the report to be created, sometimes it takes some time expecially if
   in debug mode
